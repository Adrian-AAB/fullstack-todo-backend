My To-Do-List APP API

RESTful JSON API

Retrieve all tasks

GET /tasks

Create a task

POST /task

Retrieve a specific task

GET /task/{id}

Update a specific task

PATCH /task/{id}

Attaching a JSON body with the possible below fields:

  • title (string)
  • message (string)

Delete a specific item

DELETE /task/{id}

reset database to empty

PATCH /tasks/reset
